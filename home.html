Desenvolvimento, Segurança e Operações

Um ambiente completo, secure by design, cloud native, pronto para receber sua aplicação e lógica de negócios. Utilizando todas funcionalidades extendidas do AWS AIM, conta com uma VPN de entrada, cluster Kubernetes, banco de dados relacional de sua escolha, site estático e repositório de infraestrutura como código (IaC) que ajuda implementar GitOps e CICD no seu fluxo de trabalho.
